# 폰트 파일 폴더

나눔스퀘어 라운드(Nanum Square Round) 폰트 파일을 **이 폴더(`assets/fonts/`)** 에 넣어 주세요.

## 현재 적용된 파일 (assets/fonts/)

| 파일명 | 굵기 (font-weight) |
|--------|---------------------|
| `NanumSquareRoundL.ttf` | 300 (Light) |
| `NanumSquareRoundR.ttf` | 400 (Regular) |
| `NanumSquareRoundB.ttf` | 700 (Bold) |
| `NanumSquareRoundEB.ttf` | 800 (ExtraBold) |

위 4개 파일 기준으로 메인·배틀 CSS에 @font-face가 적용되어 있습니다.
