# 골드 던전 인트로 구성

## 진행 흐름

1. **골드 던전 버튼 클릭** → 페이드 아웃
2. **페이드 인** → 골드 던전 입구 배경
3. **경비병 2명** → 좌우에서 각각 중앙으로 이동하며 등장
4. **대사창** → 올라오며 나레이션 시작
5. **경비대장** → 원내 캐릭터 등장처럼 등장
6. **대화 종료** → 경비대장 퇴장
7. **경비병 2명** → 좌우로 비켜서며 퇴장
8. **시스템 메시지** → "[ 시스템 : 골드 던전에 진입합니다. ]"
9. **대사창 터치** → 페이드 아웃
10. **던전 리스트** 출력

## 구현 상태

- **goldDungeonIntro.js**: 시나리오 구조 정의됨
- **engine.js**: `guards_enter`, `guards_step_aside` 타입 미구현
- **index.html**: goldDungeonIntro.js 미로드, showDungeon()에서 인트로 미연동

### 추가 필요 작업

1. `assets/images/gold_dungeon/` 폴더에 위 이미지 4개 배치
2. engine.js에 `guards_enter`, `guards_step_aside` 이벤트 핸들러 구현
3. index.html에 goldDungeonIntro.js 로드 추가
4. showDungeon() 수정: 골드 던전 버튼 클릭 시 인트로 재생 → 완료 후 던전 리스트 표시
